Дорогой пользователь!

Вы в одном шаге от того, чтобы получить премиальную версию приложения, название покупки: "Премиальная подписка"

Подписка будет оплачиваться ежегодно, следующий функционал будет активирован сразу после подписки
  • Поиск
  • Расширенный набор тем
  • Специальные автоматические папки (недавнее, часто просматриваемое, скоро истекает)
  • Простой логин без нажатия кнопки Вход
  • Автоматический ночной режим
  • Подсказки для пароля
  • Свои иконки для записей
  • Множество других полезных вещей, таких как смена шрифтов и другие визуальные настройки
Цена подписки отображается на кнопке покупки выше
  • Оплата будет произведена через iTunes при подтверждении покупки
  • Подписка автоматически возобновляется, если автоматическое продление не выключено по крайней мере за 24 часа до окончания текущего периода
  • Плата за продление будет взиматься за 24 часа до конца текущего периода
  • Подписка может управляться пользователем, а автоматическое обновление может быть отключено путем перехода к настройкам учетной записи пользователя после покупки
  • Любая неиспользованная часть бесплатного ознакомительного периода, если предлагается, будет аннулирована, если пользователеь пользователь приобретает подписку
